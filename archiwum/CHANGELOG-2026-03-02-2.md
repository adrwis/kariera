### 12. Font preload (H8)
**Problem:** Google Fonts CSS nie miał preload hint — browser czekał na odkrycie linku w parsowaniu HTML.
**Rozwiązanie:** Dodano `<link rel="preload" as="style">` przed `<link rel="stylesheet">` dla Google Fonts CSS URL.

| Plik | Zmiana |
|------|--------|
| `index.html` | +`<link rel="preload" ... as="style">` dla Google Fonts |

### 13. History API migration (C1)
**Problem:** Hash routing (`#/zawod/psycholog`) — brzydkie URLe, gorsze SEO, brak integracji z browser history.
**Rozwiązanie:** Pełna migracja na History API (pushState). Nowe URLe: `/kariera/zawod/psycholog`, `/kariera/wyniki?q=...`. Delegated click handler przechwytuje linki wewnętrzne. Backwards compat: hash URL → replaceState. 404.html SPA redirect via sessionStorage. SW navigation fallback. `<base href="/kariera/">` dla poprawnych ścieżek względnych.

| Plik | Zmiana |
|------|--------|
| `js/app.js` | Router: `pathname` zamiast `hash`, `popstate` zamiast `hashchange`, `go()` helper, delegated click handler, hash migration, `BASE` const, `lastResultsPath` |
| `index.html` | +`<base href="/kariera/">`, +SPA redirect script, href `#/...` → `/kariera/...` (8 popular tags + 3 back links) |
| `404.html` | Rewrite: sessionStorage SPA redirect zamiast meta refresh |
| `sw.js` | +SPA navigation fallback (serve index.html for /kariera/ subpaths), cache v1→v2 |

### 14. Minifikacja JS + CSS (H7)
**Problem:** Pliki JS + CSS serwowane bez minifikacji (~130 KB raw).
**Rozwiązanie:** Build script z esbuild `transform()` API. Wygenerowane pliki `.min.js` i `.min.css`. index.html + SW zaktualizowane na minified references. Total -43.7 KB.
